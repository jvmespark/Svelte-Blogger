# Svelte Blogger
A bare minimum and non-bloated blogger website template

## Features
- dynamic slug routing
- markdown support
### Work In Progress
- Latex support
- expanded md-html rendering components
- post comment section
- scalable design
- documentation

## Technologies  
### Frontend
- Svelte, Typescript
### Backend
- Node, Mdsvex, Disqus

## Getting Started
- git clone
- customize variables
- host your disqus server
- launch on github pages
